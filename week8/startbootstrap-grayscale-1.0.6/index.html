About Google

Google has grown. It started out as a simple search engine, but now it reaches into nearly every aspect of the technical world: computers, email, web browsers, phones, even cars.

This web/paper hybrid seeks to delve deeper into the behind-the-scenes aspects of Google and see how, exactly does a tech giant use technology.

Frameworks

Google has 4 primary frameworks that it uses.

Perl

The Perl programming language opts to go wide-it can be used for a variety of tasks, including system administration, text manipulation, web development, network programming, and more.

PHP

PHP is a general-purpose scripting language designed for web development. It can be embedded into HTML.

J2EE

J2EE, also known as Java Platform Enterprise Edition, is used for developing secure Java applications.

Adobe ColdFusion

Coldfusion is an application server and software development framework used for the development of computer software in general, and dynamic web sites in particular.
